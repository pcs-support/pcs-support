コマンド 説明 備考
🌐 ネットワーク（Windows）
ipconfig
 IPアドレスの確認 現在のIPアドレス・サブネットマスク・ゲートウェイを表示
ipconfig /all
 詳細なネットワーク情報の確認 MACアドレス・DHCPサーバーなども含めた詳細情報を表示
ping 192.168.x.x
 疎通確認 対象IPアドレスと通信できるか確認。x.xは実際のアドレスに変更
ssh root@192.168.x.x
 サーバーへのSSHリモート接続 x.xは接続先サーバーの実際のIPアドレスに変更
💾 ディスク（Linux）
df
 ディスク使用量の確認 マウントされた全パーティションの使用量をバイト単位で表示
df -h
 ディスク使用量の確認（見やすい単位） GB・MB など人が読みやすい単位で表示（-h = human-readable）
du -shc *
 各フォルダのサイズと合計の確認 カレントディレクトリ内の各フォルダサイズと総合計を表示
⚙️ システム（Linux）
top
 プロセス・CPU・メモリ使用状況の確認 リアルタイムでシステムリソースの使用状況を表示。終了は「q」キー
📂 ディレクトリ移動（Linux）
cd /○○/○○
 絶対パスでディレクトリへ移動 ○○を実際のパスに変更。例：cd /mnt/backup
cd ..
 1つ上のディレクトリへ移動 現在地の親ディレクトリに移動
cd
 ホームディレクトリへ移動 引数なしで実行するとホームディレクトリ（/root など）へ戻る
pwd
 現在のディレクトリを表示 現在いる場所のフルパスを表示
🗄️ バックアップ・RAID確認（Linux）
amCLI -l
 ディスク状態の確認 M3サーバーのみ
cat /proc/mdstat
 RAID状態の確認 M5サーバーのみ
rdiff-backup -l /mnt/backup/○○
 バックアップ一覧の確認 M3サーバーのみ　○○は対象フォルダ名に変更
該当するコマンドが見つかりませんでした。
